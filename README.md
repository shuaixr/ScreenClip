# ScreenClip

![ScreenClip](assets/icons/icons.png)

Quick screenshot tool that lives in the system tray. Hit PrintScreen, select a region, annotate, save or copy.

## Features

- PrintScreen to capture, drag to select
- Add text or rectangle annotationsgit
- Save as PNG or copy to clipboard

## Download

[Latest release](https://github.com/shuaixr/ScreenClip/releases/latest)

## Build from source

```powershell
winget install Rustlang.Rustup
cargo build --release
```

Run with `.\target\release\screenclip.exe`.

## Platforms

- **Windows** — Windows 10+
- **macOS / Linux** — planned, not there yet
